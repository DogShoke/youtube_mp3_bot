# Telegram YouTube to MP3 Downloader Bot

Этот Telegram-бот принимает ссылку на видео с YouTube, извлекает аудиодорожку, конвертирует её в формат **MP3** с битрейтом **320 kbps** и отправляет файл обратно пользователю.

## Особенности
- **Качество 320 kbps**: Скачивает аудио в наилучшем доступном формате.
- **Асинхронность**: Построен на iogram 3.
- **FFMPEG**: Автоматическая интеграция FFMPEG.
- **Автоочистка**: Скачанные файлы автоматически удаляются после отправки.

---

## Запуск в облаке (Koyeb)
Для деплоя на Koyeb укажите переменную окружения BOT_TOKEN в настройках сервиса.
